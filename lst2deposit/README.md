# lst2deposit

Creates a SIMH deposit file from an LST file created by PDP11 MACRO11 assembler.

## How To Build
Just execute make in this directory. 
```shell
make
```
See [Makefile](Makefile) for build steps.

Result of build is executable ```lst2deposit```.
